# SEM App

Frontend Vite + React para conductores y operadores del estacionamiento medido.

## Desarrollo

```bash
npm install
npm run dev
```

Requiere el backend en `http://localhost:3001`. Vite reenvía `/api` al backend (`vite.config.ts`).

Pantallas: **Inicio**, **Estacionar** (sesiones), **Cotizar**, **Activas** (listado y cobro).

## Build

```bash
npm run build
npm run preview
```
